coap13-dissector
================

Wireshark Dissector for CoAP (Draft 13)

This is a .lua dissector for Wireshark dissecting the CoAP (Contrained Application Protocol) as of Draft-13.

http://tools.ietf.org/html/draft-ietf-core-coap-13
